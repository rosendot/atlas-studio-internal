# Image Licensing

## Policy

Never use images sourced from Google Images or any site without a verified commercial license. All images used on client sites must come from an approved source below.

## Approved sources

### Paid (preferred for hero images and key visuals)
- **Shutterstock** — agency subscription. Download images here for client work. One license covers commercial use on any client site.

### Free (acceptable for supporting images)
- **Unsplash** (unsplash.com) — free, no attribution required, commercial use allowed
- **Pexels** (pexels.com) — free, no attribution required, commercial use allowed

### Client-provided photos
- Always preferred over stock — local businesses with real photos convert better
- Request photos during onboarding (see `guides/client-onboarding.md`)
- If client photos are low quality, supplement with Shutterstock in matching style

## What NOT to use
- Google Images search results
- Pinterest images
- Screenshots from other websites
- Social media images (Instagram, Facebook)
- Any image without a clear license

## Fonts

- **Google Fonts** — free, licensed for commercial web use
- **Adobe Fonts** — included with Adobe Creative Cloud if subscribed
- **Font Squirrel** — free for commercial use, verify each font's license page
- Never download fonts from random font sites — license is unverifiable
